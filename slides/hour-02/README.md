# HOUR 2 - Doing maths with numpy

In this module we'll explore how to use numpy, a numeric library that is used
in machine learning as the core library for various calculations.

This modules covers the following topics:

* What is numpy?
* Creating arrays
* Array broadcasting
* Array slicing
* Other array operations
